# Cyber Oasis Snoo: AI-Verified NFT Collection

Welcome to the **Cyber Oasis Snoo** project! This repository contains the smart contract and metadata structure for a unique collection of 10 intelligent NFTs deployed on the **GenLayer** network.

## Project Overview
This collection leverages the power of **GenLayer** to create **Intelligent NFTs (iNFTs)**. Unlike traditional NFTs, each item in the Cyber Oasis Snoo collection is built on a smart contract that enables seamless minting, listing for sale, and trading directly on-chain.

## Key Features
* **Smart Contract Development**: Built using Python for the GenLayer ecosystem, ensuring efficiency and reliability.
* **On-Chain Metadata**: All assets are stored decentralized via IPFS, ensuring permanent availability and transparency.
* **Integrated Marketplace**: The contract includes built-in functions for listing items for sale and buying them, creating a complete NFT trading ecosystem.

## Contract Information
* **Network**: GenLayer
* **Contract Address**: `0xBe9b3eD5160cd917C7FC7fdF967910c5f40a91B5`

## How it Works
1. **Deployment**: The contract is deployed on the GenLayer testnet, providing a secure foundation for the collection.
2. **Minting**: Users can mint unique NFTs by providing their metadata URL and description.
3. **Marketplace**: Owners can list their NFTs for sale, and other users can purchase them, with the contract handling ownership transfers and state updates automatically.

## Collection Structure
This collection consists of 10 unique digital assets, each with its own JSON metadata hosted on IPFS.
